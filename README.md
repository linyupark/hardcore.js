### 基金会前端代码

安装开发环境

```
  $ npm i -g riot babel-core babel-cli uglify-js gulp browser-sync --registry http://registry.cnpmjs.org
  $ npm i -D --registry http://registry.cnpmjs.org
```

启动web服务
```
  $ npm run web
```

启动自动编译，进行开发
```
  #windows
  $ npm run windev

  #mac
  $ npm run macdev
```

框架文档地址
* [riotjs](http://riotjs.com/)
* [rollup.js](http://rollupjs.org/)

Enjoy it.
